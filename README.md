# TCP Server & Client

### Overwrite
This is the demo of TCP Server & Client.

### Usage
* Generate Binary
<pre>
cd tcp_server_client_demo
make
</pre>

* Start Server
<pre>
./tcp_server < Listen Port>
</pre>

* Start Client
<pre>
./tcp_client < Server IP> < Server Port>
</pre>
